# Control_Engineering
## Process control
* 控制系统中的三个基本操作：测量、决策、执行。
* 控制系统的设计目标：对于任意的外部干扰（DV），通过调节操作变量（MV）以使被控变量（CV）维持在设定值。
* 过程控制四大参数：温度、压力、流量、液位/料位。
* 被控过程增益三部分：符号、数值与单位。

## Electrical control
* 380V是线电压，转换成相电压要除以根号3(Y型连接)
* 交流电动机与直流电动机工作原理大致相似，都是基于电磁感应原理。
* <u>电力电子逆变器</u>和<u>转子位置</u>检测器就相当于电子式换向器。
